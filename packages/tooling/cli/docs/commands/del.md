## `shc del PATH`

Removes things

```
USAGE
  $ shc del PATH

ARGUMENTS
  PATH  Path that will be remove

OPTIONS
  -d, --dryRun  List what would be deleted instead of deleting
  -f, --force   Allow deleting the current working directory and outside
  -y, --yes     Automatically answer 'Yes' to the question
```
